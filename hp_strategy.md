## HPの役割と優先度

### なぜHPが必要か
- YouTubeとSNSは「プラットフォーム依存」→ いつでも規約変更・凍結リスクあり
- HPは**自分で所有できる唯一の資産**
- SEO流入で、YouTube・SNS以外からも集客できる
- メールリスト構築 → プラットフォームに左右されない直接接触が可能

### 優先度
```
フェーズ1（今）：HPは不要、YouTube・SNSに集中
フェーズ2（登録者1,000人〜）：HP開設・SEOブログ開始
フェーズ3（登録者1万人〜）：メールリスト・コンテンツハブとして本格運用
```

**※今すぐ作らなくていい。ただし設計は今から考えておく。**

## SEO戦略（ブログ）

### 基本方針
「YouTubeで話した内容を、ブログ記事として再利用する」
→ 制作コスト最小・SEO効果最大

### ターゲットキーワード例
| キーワード | 月間検索数（目安） | 難易度 |
|-----------|------------------|--------|
| ビジネスモデル とは | 高 | 高 |
| 事業計画書 書き方 | 高 | 中 |
| 経営 本 おすすめ | 中 | 低 |
| マーケティング 基礎 | 高 | 高 |
| 起業 失敗 原因 | 中 | 低 |
| 財務諸表 読み方 入門 | 中 | 低 |

### 記事作成フロー
```
YouTube動画を収録
  ↓
文字起こし（自動ツール活用）
  ↓
ブログ記事に整形・加筆
  ↓
動画を記事内に埋め込み
  ↓
公開 → SNSで告知
```

## メールリスト戦略

### なぜメールリストが重要か
- YouTube・SNSが使えなくなっても連絡できる
- ファンとの直接接触 → 案件・商品販売に活用できる
- 開封率はSNSのリーチより高い（20〜30%）

### 構築方法
1. HPに「無料特典」と引き換えにメール登録フォームを設置
2. 無料特典例：「経営者が読むべき本10冊リスト（PDF）」など
3. 週1回のメールマガジン配信（YouTube更新情報 + 独自コンテンツ）

### 使用ツール（推奨）
- **Mailchimp**（無料枠あり・日本語対応）
- **ConvertKit**（クリエイター向け・英語）
- **Benchmark Email**（日本製・使いやすい）
